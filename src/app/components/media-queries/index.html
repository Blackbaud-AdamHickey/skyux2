The media query service allows users to subscribe to screen size changes at different breakpoints. Suscribes to screen size changes. The listener function will be called whenever breakpoints change. It will be called with a SkyMediaBreakpoints argument, which is an enum that represents the new breakpoint. Its possible values are xs (screen width 767px or less), sm (screen width 768px - 991px), md (screen width 992px - 1199px), or lg (screen width 1200px or greater).
